# PULSO
**Plataforma de Unificação, Logs e Sucesso Operacional**

O PULSO é uma ferramenta de gestão estratégica projetada para o acompanhamento do Customer Success (CS) da IPNET. Ele soluciona a dor de informações fragmentadas centralizando a governança de contratos, a visibilidade de execução de serviços (Workshops, Assessments, Treinamentos) e o engajamento do cliente, fornecendo uma verdadeira Visão 360 e identificação proativa de riscos de churn.

## 💻 Stack Tecnológica
- **Frontend:** React 19, Vite, React Router DOM, Recharts
- **Backend:** Node.js, Express.js
- **Banco de Dados:** SQLite (`csm.db`)

## 🚀 Como rodar o projeto localmente
Para iniciar a aplicação em seu ambiente de desenvolvimento:
```bash
npm install
npm start
```
